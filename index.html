Geospatial Companies

The main goals behind the creation of this interactive Webmap are:

  1. For the completion of project work as a course work of Copernicus Master in Digital Earth

  2. To provide a simple interface for Geospatial Professionals to identify several Geospatial Companies available in the World

If you are intersted in finding a new job or just to get to know some geospatial companies around the world, this map is for you!

How to use this map

This Web map composed of Three Main Sections divided as Side Bar;Target Form and Map View.

SideBar

The bar has multiple tabs dedicated to delivering information regarding the Geospatial Portal in words.

About tab gives an overview of the target behind the creation web map and provides the user with a Geospatial form for logging in the Geospatial Companies they are dedicated (if and only if it is officially recognized).

Help tab will familiarize users with an overview of functionalities available in the web map.

Query tab has been integrated into the side to provide the filtering functionality in the map. User can query based on Domain of the company and Size of the company, so it is mandatory for a user who logs in new companies to fill up these two fields along with the name of the company. To find the companies based on these two or single filter user can choose the values from the drop-down box located left of the Field Name.

Cartographers tab gives a short introduction to the developers and leads to their website.

Target Form

ArcGIS GeoForm is integrated into the Portal for allowing a user to Log in new companies other than those available in the map view. But no user will be allowed to make any changes to the data and in case of need will have to contact the concerned person. The companies logged in this form will be added to the main database and thus displayed in the map view. The location in the GeoForm is requested to be pinned accurately. Only Officially Registered Companies Should be Logged in.

Map View

The Map View shows the Geospatial Companies Located Worldwide. Various functionality has been integrated here that includes:

GeoCoder: For Searching an Address WorldWide.

ESRI WFS: For embedding ESRI Web Feature Service. Data has been integrated as WFS.

FeatureHighlight: When a mouse cursor is moved over the feature icon the point of interest will be highlighted to avoid confusion.

EventListener: This functionality will allow a user to know the coordinate of a location when double-tapped anywhere on the map view.

CountrySelect: It will allow a user to zoom to the selected country location.

Home: Home icon at the top right corner in the map view allows user to get back o the default map extent.
